本プライバシーポリシーは、就活ES通過くん（以下「本サービス」といいます。）におけるユーザー情報の取扱いを定めるものです。

1. 取得する情報

  • 本サービスは、ユーザーの同意または利用操作に基づき、次の情報を取得する場合があります。
  • Googleアカウント等の認証に関する情報：メールアドレス、氏名または表示名、プロフィール画像、認証ID等。
  • プロフィール情報：大学、専攻、志望業界、職種、その他ユーザーが入力した背景情報。
  • ガクチカ・ES作成情報：経験内容、音声入力またはテキスト入力の内容、AIからの質問への回答、企業名、企業メモ、ES設問、目標文字数、補足情報、生成された文章、再生成履歴等。
  • 利用状況情報：ログイン状況、利用日時、画面遷移、生成回数、AI利用量、トークン使用量、エラー情報、デバッグ情報、紹介コード、紹介割引の状態等。
  • 決済関連情報：プレミアムプランの購読状態、決済事業者から連携される顧客ID、サブスクリプションID、請求状態、割引状態等。ただし、クレジットカード番号等の詳細な決済情報は、本サービスでは保持せず、決済事業者が管理します。
  • 端末および通信に関する情報：IPアドレス、ブラウザ、OS、リファラ、Cookie、ローカルストレージ、セッション情報その他本サービスの提供に必要な技術情報。

2. 利用目的

  • 本サービスは、取得した情報を次の目的で利用します。
  • 本サービスの提供、本人確認、ログイン管理、ユーザー画面の表示、データ保存、AIによる整理・生成・再生成、音声の文字起こし、生成結果の表示および管理のため。
  • プレミアムプラン、決済、請求、解約、紹介割引、利用制限、利用回数管理、利用権限確認のため。
  • 不具合調査、品質改善、AI生成品質の確認、利用状況の分析、セキュリティ対策、不正利用防止、過度な負荷の検知および対応のため。
  • 問い合わせ対応、重要なお知らせ、規約やポリシー変更の通知、法令上必要な対応のため。
  • 本サービスの改善、新機能の検討、利用体験の向上、運営判断のため。

3. AI処理および外部サービスの利用

  • 本サービスは、AIによる文章生成、音声文字起こし、認証、データ保存、決済、ホスティング、分析、不具合調査等のため、外部サービスを利用する場合があります。
  • 利用する外部サービスには、Supabase、OpenAI、Stripe、Google、Vercelその他本サービスの提供に必要なサービスが含まれる場合があります。
  • ユーザーが入力した文章、音声、企業情報、ES設問、補足情報、生成履歴等は、本サービスの提供に必要な範囲で、AI処理または外部サービスに送信される場合があります。
  • 外部サービスにおける情報の取扱いは、各外部サービスの規約およびプライバシーポリシーにも従います。

4. 第三者提供

  • 本サービスは、次の場合を除き、ユーザーの個人情報を第三者に提供しません。
  • ユーザーの同意がある場合。
  • 本サービスの提供、AI処理、認証、決済、データ保存、問い合わせ対応、セキュリティ対策等に必要な範囲で、外部サービスまたは委託先に提供する場合。
  • 法令に基づく場合、裁判所、行政機関、捜査機関等から適法な要請を受けた場合。
  • 人の生命、身体、財産の保護その他法令上認められる場合。

5. 安全管理

  • 本サービスは、取得した情報について、漏えい、滅失、毀損、不正アクセス、不正利用を防止するため、合理的な安全管理措置を講じます。
  • ただし、インターネット通信、外部サービス、AI処理、ユーザー端末、認証情報の管理には固有のリスクがあり、完全な安全性を保証するものではありません。

6. 保存期間および削除

  • 本サービスは、利用目的の達成、契約管理、問い合わせ対応、不正利用防止、法令対応、会計・請求管理、運営上必要な範囲で、ユーザー情報を保存します。
  • ユーザーから削除、退会、開示、訂正、利用停止等の請求があった場合、本人確認のうえ、法令および運営上必要な範囲で対応します。

  • ユーザーは、ブラウザの設定によりCookie等を制限できますが、その場合、本サービスの一部機能が利用できない場合があります。
